เกี่ยวกับระบบ About the System

ระบบนี้พัฒนาขึ้นเพื่อช่วยประเมินระดับการออกเสียงภาษาอังกฤษสำหรับผู้เรียนชาวไทยที่เรียนภาษาอังกฤษในฐานะภาษาต่างประเทศ (Thai EFL learners) โดยใช้โมเดล Machine Learning ที่ผ่านการฝึกอบรมมาอย่างดี

This system was developed to help assess the English pronunciation level of Thai learners who study English as a Foreign Language (Thai EFL learners) using a well-trained Machine Learning model.

วัตถุประสงค์ Objectives

  1. เพื่อพัฒนาเว็บแอปพลิเคชันที่รองรับการอัพโหลดและประเมินไฟล์เสียงคำหรือประโยคภาษาอังกฤษ 1-10 ไฟล์พร้อมกัน To develop a web application that supports uploading and assessing 1-10 English word or sentence audio files simultaneously
  2. เพื่อสร้างโมเดล Machine Learning สำหรับประเมินระดับการออกเสียงตามเกณฑ์ 3 ระดับ (สูง, กลาง, ต่ำ) To create a Machine Learning model for assessing pronunciation levels according to 3 criteria (High, Mid, Low)
  3. เพื่อรวมระบบเว็บแอปพลิเคชันและโมเดลให้ทำงานร่วมกันได้อย่างราบรื่น To integrate the web application system and model to work together seamlessly
